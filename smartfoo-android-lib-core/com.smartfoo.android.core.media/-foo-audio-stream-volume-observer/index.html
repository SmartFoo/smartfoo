FooAudioStreamVolumeObserver

open class FooAudioStreamVolumeObserver

Observes changes to audio stream volumes and notifies registered callbacks.

Registers a android.database.ContentObserver on CONTENT_URI to detect volume changes for one or more audio stream types. Multiple listeners may be attached for the same or different stream types; the content observer is registered lazily on the first attach call and unregistered after the last detach call.
