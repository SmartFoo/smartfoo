open class FooLongSparseArray<E> : Cloneable
Mutation of androidx.collection.LongSparseArray that changes the following:
  • delete calls remove, instead of the other way around
  • remove returns the removed value (or null)
  • removeAt returns the removed value (or null)
  • put returns the index returned from binarySearch
  • Adds iterateKeys
  • Adds iterateValues
  • throws IllegalArgumentException for null values
  • Extra debug logging
One note to the original code's below comment is that the advantage of using this type of collection is that the keys are always sorted!
