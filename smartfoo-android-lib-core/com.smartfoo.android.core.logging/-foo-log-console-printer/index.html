FooLogConsolePrinter

open class FooLogConsolePrinter : FooLogPrinter

A FooLogPrinter that writes formatted log messages to out.

Delegates formatting to a supplied FooLogFormatter. Useful for unit tests, desktop JVM contexts, or any environment where LogCat is unavailable. Use getInstance to obtain or create the shared singleton.
