SetLogLimitDialogFragment

open class SetLogLimitDialogFragment : DialogFragment

A dialog fragment that lets the user configure two log size limits via seekbars: the overall log size limit (0–1024 KB) and the email-attachment log size limit (0–128 KB).

The host android.app.Activity must implement SetLogLimitDialogFragmentCallbacks. Create instances via newInstance and show them through the fragment manager.
