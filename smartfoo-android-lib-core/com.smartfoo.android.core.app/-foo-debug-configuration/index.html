FooDebugConfiguration

interface FooDebugConfiguration

Configuration contract for the debug log activity.

Implementations persist these settings (e.g. in android.content.SharedPreferences) and expose them to FooDebugActivity so that log-size limits, email limits, and file-logging state survive across sessions.
