FooBoolean

object FooBoolean

Conversion utilities for boolean values across multiple representations.

Supports converting to/from Boolean, Byte, Number, and String. The string conversion recognises numeric values as well as the literals "true", "false", "yes", "no", "y", and "n" (case-insensitive).
