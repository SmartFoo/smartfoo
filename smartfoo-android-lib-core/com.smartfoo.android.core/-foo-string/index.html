FooString

object FooString

String manipulation, formatting, and conversion utilities.

Covers null-safe checks, hex/bit string formatting, null-terminated byte array encoding, time-duration formatting, camel-case splitting, and SpannableString creation for Android text views. UTF-8 is the assumed character set for all byte–string conversions.
