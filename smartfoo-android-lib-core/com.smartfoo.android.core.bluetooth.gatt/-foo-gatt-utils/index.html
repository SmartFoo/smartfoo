FooGattUtils

open class FooGattUtils

Static utility methods for GATT-level Bluetooth LE operations.

Includes device-address conversion between string and long forms, safe wrappers around disconnect and close that handle known exceptions, connection state name formatting, and byte-array serialisation helpers that mirror the hidden android.bluetooth.BluetoothGattCharacteristic encoding methods. This class is not instantiable.
