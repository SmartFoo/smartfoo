connectGatt

@RequiresPermission(value = Manifest.permission.BLUETOOTH_CONNECT)
open fun connectGatt(remoteDevice: BluetoothDevice, autoConnect: Boolean, bluetoothGattCallback: BluetoothGattCallback): BluetoothGatt

Initiates a GATT connection to the remote device, working around a race condition in Android 6.x and earlier where the auto-connect flag was not properly propagated before the native connectGatt call.

On Android 7.0 (N) and above, or when autoConnect is false, this method delegates directly to the platform connectGatt API. On older versions with autoConnect = true it uses reflection to set the flag before connecting, falling back to the standard API if reflection fails.

Return

the BluetoothGatt connection object, or null if the connection could not be initiated

Parameters

remoteDevice

the device to connect to; returns null if null

autoConnect

true to use the OS background auto-connect mechanism

bluetoothGattCallback

callback to receive GATT events
