# Package com.smartfoo.android.core.crypto

Cryptography utilities. `FooCrypto` provides static helpers for hashing (MD5 proprietary hash, SHA-256), HMAC-SHA-256 message authentication, AES/ECB symmetric-key operations, and secure-random byte generation. Algorithm name constants (`AES`, `HMACSHA256`, `SHA256`) are exposed as named fields to avoid magic strings throughout the codebase.
